ĐỀ THI ACSL

What Does This Program Do?

Hướng dẫn: Đọc kỹ chương trình giả (pseudocode ACSL) và xác định kết quả hoặc chức năng của chương trình.

Câu 1

Đầu vào: h = 50, r = 10

input h, r
b = 0
if h > 48 then
    b = b + (h - 48) * 2 * r
    h = 48
end if
if h > 40 then
   b = b + (h - 40) * (3/2) * r
   h = 40
end if
b = b + h * r
output b

Câu 2

a = "BANANAS"
num = 0; t = ""
for j = len(a)-1 to 0 step -1
     t = t + a[j]
next j
for j = 0 to len(a) - 1
    if a[j] == t[j] then
       num = num + 1
    end if
next j
output num

C = 0; S = 0
FOR I = 1 TO 50
  IF I / 2 = INT(I / 2) THEN
     C = C + 1
  ELSE
     S = S + 1
  END IF
NEXT I
PRINT C, S

Câu 5

a = 12: b = 1: c = 0: d = 4: e = 2
IF a > d THEN a = a - d
IF (d - b) < (e - a) THEN d = d + e
IF a * b == d * e THEN
   e = a * b / e
ELSE
   d = d * e / a
END IF
IF d ^ 2 <= (b + 1) ^ 2 THEN
   d = b + 1
ELSE
   b = b + 1
END IF
IF a + b * c == d + e * c THEN
   a = b * c
ELSE
   d = e * c
END IF
output (a + e) / b + (d + c) ^ b * c

Lưu ý: Toán tử ^ hoặc ↑ biểu diễn lũy thừa trong ACSL.
